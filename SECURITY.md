# Security Policy

## Reporting a Vulnerability
If you believe you have found a security vulnerability, please report it privately.

- Email: contact@qryptalabs.com
- Subject: [SECURITY] QRYPTA Contracts Vulnerability Report

Please include:
- Description and impact
- Steps to reproduce / PoC (if available)
- Affected functions/areas
- Suggested mitigations (optional)

Do not disclose the issue publicly until we have had a chance to investigate and release a fix.
